# Event Mapper

## install packages

```
docker-compose run --rm front sh -c "yarn add axios"
```
```
docker-compose run --rm front sh -c "yarn add bulma"
```
```
docker-compose run --rm front sh -c "yarn add leaflet"
```

## start containers
```
docker-compose up -d
```
* backend
  * http://localhost:8000/
* frontend
  * http://localhost:3000/
